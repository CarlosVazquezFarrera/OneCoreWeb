# OneCoreWeb
 Proyecto web hecho con Angular 
 
**El proyecto tienen un CRUD completo, en el que se conecta a un Api desarrollada con net core:**
- [ ] Login
- [ ]  Consulta de usuarios
- [ ]  Editar usuarios
- [ ]  Eliminar usuarios
